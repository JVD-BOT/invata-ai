# ÎnvațăAI — Ghid AI în Limba Română

Romanian-language AI tools directory, tutorials, and education portal.

## Features

- **AI Tools Directory** — Curated reviews of AI tools with ratings, pricing, and categories
- **Blog & Tutorials** — Practical articles in Romanian about using AI
- **Newsletter** — Weekly email signup
- **Category Filtering** — Browse articles by topic
- SEO-optimized for Romanian AI search queries
- Editorial design, mobile-responsive
- Zero backend — free hosting on Vercel

## Deploy for FREE

1. Create a new GitHub repository called `invata-ai`
2. Upload all these files to it
3. Go to vercel.com → Add New → Project → Import `invata-ai`
4. Click Deploy — live in 60 seconds!

## Local Development

```bash
npm install
npm run dev
```
